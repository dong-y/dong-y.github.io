Introduction

This is a tutorial on how to create a free blog with Octopress and Github for greenhands, well just like me! I am a PhD student in Information Systems and would like to share what I learned with others through a blog. But I know little about html and web development… markdown really saved my life. If you are interested in coding a blog like me, this tutorial might be helpful.
